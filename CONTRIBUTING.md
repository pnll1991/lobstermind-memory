# Contributing to LobsterMind Memory

Thanks for considering contributing! Here are some guidelines:

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/lobstermind-memory.git`
3. Install dependencies: `npm install`
4. Build: `npm run build`

## Development

- **TypeScript**: All code is in TypeScript
- **Build**: Run `npm run build` before testing
- **Testing**: Manual testing via OpenClaw CLI commands

## Code Style

- Use TypeScript strict mode
- Follow existing patterns in `index.ts`
- Add console logs for debugging (prefixed with `[lobstermind]`)
- Keep functions small and focused

## Pull Requests

1. Create a feature branch
2. Make your changes
3. Test thoroughly
4. Open a PR with a clear description

## Issues

- Use issues for bugs and feature requests
- Include steps to reproduce for bugs
- Be specific about desired behavior for features

## Security

- Never commit user data, database files, or credentials
- Report security issues privately
- Block sensitive data storage (emails, passwords, etc.)

## Questions?

Open a discussion or reach out via the OpenClaw community.
